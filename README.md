# 3D Hartwig Chess Set

A 3D chess game built entirely with HTML, CSS, and JavaScript featuring the iconic Bauhaus-style Hartwig chess pieces.

## Description

This project is a fully playable 3D chess game implemented using CSS3 transforms and HTML/JavaScript. It features a beautiful recreation of Josef Hartwig's famous Bauhaus chess set design from 1924, where pieces are represented by geometric shapes. The game includes full chess logic with move validation, check/checkmate detection, and multiple visual themes.

## Features

- 3D rendered chess board and pieces using pure CSS3 transforms
- Multiple visual themes (Classic, Marble, Flat, Wireframe)
- Full chess rules with move validation
- Check and checkmate detection
- Drag and drop piece movement with touch support
- Move undo functionality
- Rotatable board view for each player's turn
- Optional board frame/border display

## Technologies Used

- HTML5
- CSS3 (3D transforms, animations)
- JavaScript (ES5)
- jQuery 1.7.2
- [Photon](http://photon.attasi.com) - CSS 3D lighting effects
- [Chess.js](https://github.com/jhlywa/chess.js) - Chess move generation and validation
- [Prefix-free](https://leaverou.github.io/prefixfree/) - CSS prefix handling

## Installation

```bash
# Clone the repository
git clone https://github.com/bryanseah234/Chess3D.git

# Navigate to project directory
cd Chess3D

# Open index.html in a WebKit browser (Chrome, Safari)
```

No build process or dependencies installation required - just open `index.html` in your browser.

## Usage

```bash
# Open in a WebKit-based browser (Chrome, Safari, etc.)
open index.html
# or simply double-click index.html
```

1. Click "Play" to start the game
2. Select "Themes" to change the visual style
3. Drag and drop pieces to make moves
4. Valid moves are highlighted on the board
5. Use the undo button to take back moves
6. The board automatically rotates to show the current player's perspective
